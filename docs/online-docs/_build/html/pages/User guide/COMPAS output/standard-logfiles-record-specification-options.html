Program option propertiesÂ¶

When specifying known properties in a log file record specification record, the property name must be prefixed with the property type. Currently there is a single binary property type available for use: PROGRAM_OPTION.

For example, to specify the program option property RANDOM_SEED, use:

PROGRAM OPTION::RANDOM_SEED

Following is an alphabetical list of program option properties available for inclusion in log file record specifiers.
